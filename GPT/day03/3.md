# 1. 很多变成语言的索引都是从0开始，python也不例外，从0开始计算偏移量
# 2. 会取到实际第二个到第四个索引，也就是1,2,3
# 3. 等价
# 4. 表示取最末尾的那个元素
# 5. 索引表示元素的下标，一个元素对应一个下标，切片是根据索引从字符串内切出来的实际内容
# 6. 因为len(text)计算的是实际长度，比如有10个字符，但不存在text[10]，因为实际索引是从0到9
