## 学习目标

完成后需要确认你是否能够：

1. 创建并运行 `.py` 文件。
2. 正确使用注释、缩进、变量、`print()` 和 `input()`。
3. 理解 `input()` 得到的数据类型。
4. 完成基本类型转换和简单运算。
5. 使用清晰的变量名组织代码。

## 核心知识点

### 1. 程序执行

Python 通常按照代码从上到下依次执行。

```python
print("第一步")
print("第二步")
```

### 2. `input()` 的返回值

无论用户输入的是数字还是文字，`input()` 默认返回的都是字符串 `str`。

需要计算时，通常要进行类型转换：

```python
age_text = input("请输入年龄：")
age = int(age_text)
```

### 3. 变量命名

推荐使用小写字母和下划线：

```python
user_name = "Tom"
test_case_count = 20
```

不推荐：

```python
a = "Tom"
TestCaseCount = 20
```

变量名不能以数字开头，也不能使用 Python 关键字。
