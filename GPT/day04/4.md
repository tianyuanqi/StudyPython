字符串格式化
1. 为什么需要字符串格式化
之前你可能这样输出：
name = "login"
status_code = 200
print("接口名称:", name, "状态码:", status_code)

这种方式可以使用，但内容复杂时不容易控制格式。
字符串格式化可以把变量直接嵌入字符串：
message = f"接口名称：{name}，状态码：{status_code}"
print(message)
前面的 f 表示这是一个 f-string。



2. f-string 基础语法
name = "yuanqi"
age = 28
print(f"姓名：{name}，年龄：{age}")
花括号中不仅能写变量，也能写简单表达式：

price = 29.9
quantity = 3
print(f"总价：{price * quantity}")
现阶段建议复杂计算先保存到变量，再格式化输出，代码会更清晰。

3. 保留小数位
response_time = 123.45678
print(f"响应时间：{response_time:.2f} ms")
其中：.2f 表示：
f：按浮点数格式显示
.2：保留两位小数

例如：
value = 3.1415926
print(f"{value:.1f}")
print(f"{value:.2f}")
print(f"{value:.4f}")
注意：格式化显示会进行四舍五入，但不会修改原变量。

4. 百分比格式化
passed_cases = 80
executed_cases = 100
pass_rate = passed_cases / executed_cases

print(f"通过率：{pass_rate:.2%}")

.2% 表示：

自动乘以 100
添加 %
保留两位小数

结果类似：

通过率：80.00%

因此不要在计算时自己再乘一次 100。

错误思路：

pass_rate = passed_cases / executed_cases * 100
print(f"{pass_rate:.2%}")

这样会再次乘以 100，结果会被放大。

5. 整数补零

测试用例编号经常需要固定宽度：
case_number = 7
print(f"CASE_{case_number:03d}")
结果：
CASE_007
其中：
03d
表示：
d：按整数显示
宽度为 3
不足部分用 0 补齐

例如：

case_number = 12

会得到：

CASE_012
6. 对齐和宽度
name = "login"

print(f"|{name:<10}|")
print(f"|{name:>10}|")
print(f"|{name:^10}|")

含义：

<：左对齐
>：右对齐
^：居中
10：总宽度

这在打印简单测试报告时很有用。

7. 花括号本身如何输出
f-string 中花括号有特殊作用。
需要真正输出 { 或 } 时，要写两个： print(f"请求数据：{{'username': 'admin'}}")
